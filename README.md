# Vrinda-store-excel-dashboard
Excel Dashboard project analyzing sales performance for Vrinda Store (2022)
This project analyzes Vrinda Store's annual sales performance for 2022 using Excel dashboards.
The report includes insights on orders, channels, customer demographics, and top-performing states.
 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Power Query
- Slicers
- Data Cleaning

Key Insights
- Women contribute 64% of sales.
- Amazon is the top sales channel (35%).
- Maharashtra generates the highest revenue.
- Sales peak during March and April.
